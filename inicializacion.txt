Hola papus. Esta es una guia para inicializar el proyecto por si la ocupan.
    - Para empezar deben de bajar el proyecto de github a su local, si no lo han hecho.
    - Una vez instalado deberan instalar el proyecto con el comando "npm install" en la raiz del proyecto.
    - Por ultimo, para ejecutar el proyecto solo es necesario usar "npx expo start" en una terminal.
